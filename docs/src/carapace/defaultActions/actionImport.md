# ActionImport

[`ActionImport`] parses the json generated by [Export] and imports it as [Action].

```go
carapace.ActionImport([]byte(`
{
  "version": "unknown",
  "messages": [],
  "nospace": "",
  "usage": "",
  "values": [
    {
      "value": "first",
      "display": "first"
    },
    {
      "value": "second",
      "display": "second"
    },
    {
      "value": "third",
      "display": "third"
    }
  ]
}
`))
````
