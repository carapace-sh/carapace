# ActionImport

[`ActionImport`] parses the json generated by [Export] and imports it as [Action].

```go
carapace.ActionImport([]byte(`
{
  "Version": "unknown",
  "Nospace": "",
  "RawValues": [
    {
      "Value": "first",
      "Display": "first"
    },
    {
      "Value": "second",
      "Display": "second"
    },
    {
      "Value": "third",
      "Display": "third"
    }
  ]
}
`))
````
